## Historial de Speedruns
<Tabs>
    <TabItem label="The Journey">
        <Steps>
            1. Shin Megami Tensei: Persona 3 FES se lanzó en Japón el 19 de abril de 2007.
            2. La primera run fue hecha por newby24 con un tiempo de [**43:49:01**](https://www.twitch.tv/videos/157810640).
            3. Un año después, el 31/07/2018, Darkkefka batió el récord mundial con un tiempo de [**23:12:09**](https://youtu.be/uLJzJFE-MDU)
            4. El 24/05/2020, Keeanolookalike redujo aún más el tiempo a [**21:55:03**](https://www.twitch.tv/videos/630455571)
            5. El 23/10/2021, Darkkefka estableció un nuevo récord con un tiempo de [**20:17:44**](https://youtu.be/VPmV0vBxJXs)
            6. El 4/09/2022, TatsuyaAct logró el primer sub 20h con un tiempo de [**19:50:47**](https://youtu.be/bxJ3uwRKp9k)
            7. El 26/02/2023, Fenr1r79 estableció un nuevo récord con un tiempo de [**19:26:33**](https://www.twitch.tv/videos/1749735432)
            8. El actual poseedor del world record es Darkkefka con un tiempo de [**19:23:14**](https://www.twitch.tv/videos/1963205251)
        </Steps>
    </TabItem>
    <TabItem label="The Answer">
        <Steps>
            1. Shin Megami Tensei: Persona 3 FES se lanzó en Japón el 19 de abril de 2007.
            2. La primera run fue hecha por lilfatkid con un tiempo de [**13:56:24**](https://www.twitch.tv/videos/257118140). Mejoraría el récord hasta un tiempo de [**9:09:39**](https://youtu.be/qt9UkUCJ-aM)

            5. El 21/09/2022, Kap fue la segunda persona en lograr un tiempo inferior a 8 horas y estableció un nuevo récord con un tiempo de [**7:48:47**](https://youtu.be/cU0C2-jz2dA)
            2. El actual poseedor del récord mundial es Kap con un tiempo de [**7:21:37**](https://youtu.be/hy9BOzJkwIA)
