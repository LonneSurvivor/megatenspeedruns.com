## Historial de Speedruns
<Tabs>
    <TabItem label="Any% - Easy - PC">
        <Steps>
            1. Persona 5 Strikers se lanzó mundialmente el 22 de febrero de 2021.
            2. La primera run Fue realizado por TatsuyaAct con un tiempo de [**5:08:17**](https://www.twitch.tv/videos/925716221). Lo mejoraría a un tiempo de [**4:25:29**](https://www.twitch.tv/videos/990207868).
            3. El 19/07/2024, Rodillo establecería un nuevo récord con un tiempo de [**4:25:03**](https://youtu.be/hJTo_LXFQFc). Mejoraría el récord a un tiempo de [**4:16:40**](https://www.twitch.tv/videos/2217608430).
            2. El actual poseedor del récord mundial es TatsuyaAct con un tiempo de [**4:10:59**](https://youtu.be/hSTqwU9EVxo).
        </Steps>
    </TabItem>
    <TabItem label="Any% - Easy - Console">
        <Steps>
            1. Persona 5 Strikers se lanzó mundialmente el 20 de febrero de 2020.
            2. La primera run la realizó Worldie con un tiempo de [**7:27:38**](https://www.twitch.tv/videos/796515294). El disminuyó el tiempo a [**5:07:33**](https://youtu.be/Rx5Ah09U2Hk).
            3. El 23/07/2022, Jordiee estableció un nuevo récord con un tiempo de [**4:57:47**](https://youtu.be/8fHAQq8vnJY)
            4. El actual poseedor del récord mundial es Jordiee con un tiempo de [**4:37:52**](https://youtu.be/wDxsbOBD1H8)
        </Steps>
    </TabItem>
    <TabItem label="Any% - Hard - PC">
        <Steps>
            1. Persona 5 Strikers se lanzó mundialmente el 22 de febrero de 2021.
            2. El primer tiempo registrado lo realizó TatsuyaAct con un tiempo de [**9:39:07**](https://www.twitch.tv/videos/767958327) el 11/10/2020. Cuatro meses después, lo mejoró a [**7:35:36**](https://www.twitch.tv/videos/941146062).
            3. El 20/03/2021, HeroMystic estableció un nuevo récord con un tiempo de [**6:25:29**](https://youtu.be/BD11kGD5Jos). 
            4. El actual récord mundial lo tiene HeroMystic con un tiempo de [**6:10:32**](https://youtu.be/FF_drbeWdWs).
        </Steps>
    </TabItem>
    <TabItem label="Any% - Hard - Console">
