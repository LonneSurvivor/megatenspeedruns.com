## Historial de Speedruns
<Tabs>
<TabItem label="Hard, True Ending - Legacy">
        <Steps>
            1. Persona 5 Royal se lanzó en Japón el 31 de octubre de 2019.
            2. La primera run fue hecho por Liv con un tiempo de [**19:42:24**](https://www.twitch.tv/videos/519987799).
            3. 18 días después, Darkkefka logró un mejor tiempo siendo este de [**17:51:39**]. El VOD ya no está disponible.
            4. El 15/01/2020, Liv batió el récord anterior con un tiempo de [**17:16:34**](https://www.twitch.tv/videos/537233376). Lo mejoraría dos meses después con un tiempo de [**16:22:06**](https://www.twitch.tv/videos/563143319)
            5. El 14/04/2020, Darkkefka estableció un nuevo récord logrando un tiempo de [**16:10:53**](https://youtu.be/OGDtSBgVSjE)
            6. El 3/05/2020, Liv batió el récord mundial con un tiempo de [**15:53:24**](https://youtu.be/htEGEEuRWpU)
            7. Un día después, Darkkefka recuperó el world récord con un tiempo de [**15:38:38**](https://youtu.be/5h2_Vq7UtYI)
            8. Mayo fue un mes muy movido para el speedrunning de P5R, ya que Liv continuó con el ir y venir 14 días después y consiguió el récord con un tiempo de [**15:35:36**](https://youtu.be/ssAz7xBdnok)
            9. 8 días después, HeroMystic entró al mes de mayo de P5R al reclamar el récord con un tiempo de [**15:33:34**](https://www.twitch.tv/videos/633201183)
            10. El 1 de julio de 2020, Darkkefka recuperó el récord con un tiempo de [**15:09:45**](https://youtu.be/gPIX_rrwia0)
            11. 4 meses después, Liv recuperó el récord mundial logrando un tiempo de [**15:09:33**](https://youtu.be/vcYtGEngk_4). Lo mejoraría con la primera run de menos de 15 horas. [**14:53:29**](https://youtu.be/S-hxM0GbWtU)
            12. El 23/12/2020, Darkkefka fue el segundo speedrunner en lograr una run de menos de 15 horas y obtuvo el récord mundial con un tiempo de [**14:45:13**](https://youtu.be/AzRbQbCwY5c), Y así llegó el año de P5R a su fin.
            13. Pero Liv comenzó el 2021 sorprendiendo a todos con un nuevo récord, logrando un tiempo de [**14:39:51**](https://youtu.be/NrfB4tHgLHQ)
            2. La actual poseedora del récord mundial es Liv con un tiempo de [**14:09:58**](https://youtu.be/j8M8yMHIpdQ)
        </Steps>
    </TabItem>
    <TabItem label="Hard, True Ending - Remaster">
        <Steps>
            1. Persona 5 Royal fue remasterizado y lanzado mundialmente el 21 de octubre de 2022.
            2. La primera fue hecha por Liv con un tiempo de [**12:11:28**](https://youtu.be/Qn0pPKBRFEc)
            3. La actual poseedora del récord mundial es Liv con un tiempo de [**11:57:07**](https://youtu.be/ldfjoi0RdeE)
        </Steps>
    </TabItem>
    <TabItem label="Hard, True Ending - PC">
        <Steps>
            1. Persona 5 Royal fue remasterizado y lanzado mundialmente el 21 de octubre de 2022.
            2. La primera run fue hecha por hoxi en un tiempo de [**13:44:42**](https://www.twitch.tv/videos/1643546285)
            2. El 10/11/2022, TatsuyaAct estableció un nuevo récord con un tiempo de [**11:14:11**](https://youtu.be/L6QYXdr_aWM)
            3. El 27/06/2024, Sprayz estableció un nuevo récord con un tiempo de [**11:08:30**](https://youtu.be/Tip1uVLYavs)
            4. El actual poseedor del récord mundial es Sprayz con un tiempo de [**10:46:11**](https://youtu.be/1Xw9eA0Lp60)
        </Steps>
    </TabItem>
</Tabs>
