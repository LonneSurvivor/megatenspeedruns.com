        </Steps>
    </TabItem>
    <TabItem label="PSP - SEBEC">
        <Steps>
            1. Shin Megami Tensei: Persona se lanzó en Japón el 29 de abril de 2009.
            2. La primera run registrada fue realizada por Liv basándose en la ruta PS1 SEBEC de Supergamer57, que logró un tiempo de [**6:05:32**](https://youtu.be/7OyKC3M5TRs)
            3. El 12/06/2019, Tick_Show desarrolló la ruta Mabufu Stone y logró un tiempo de [**3:59:53**](www.nicovideo.jp/watch/sm35250359)
            4. El 26/07/2019, el speedrunner escaroon logró la primera run sub 3h con un tiempo de [**2:54:13**](https://youtu.be/VXMjtP7fWMo)
            5. El actual récord mundial de consola le pertenece a Tick_Show con un tiempo de [**2:51:05**](https://youtu.be/HWlsj6S-7Xw) y el tiempo más bajo jamás alcanzado, que se hizo en el emulador, lo tiene Fenr1r74 con un tiempo de [**2:44:48**](https://www.twitch.tv/videos/1825596953)
        </Steps>
    </TabItem>
    <TabItem label="PSP - Snow Queen Quest">
        <Steps>
            1. Shin Megami Tensei: Persona se lanzó en Japón el 29 de abril de 2009.
            2. La primera run registrada fue realizada por escargoon el 24/12/2013, completando el True Ending, con un tiempo de [**1:56:33**](https://youtu.be/ijDyyVHEmro). True Ending es la categoría menos popular y su tiempo más rápido ahora lo tiene Tick_Show con un tiempo de [**1:48:27**](www.nicovideo.jp/watch/sm35246032)
            3. La primera run registrada del Bad Ending fue realizada por Symm el 5/03/2016 con un tiempo de [**1:30:50**](https://www.twitch.tv/videos/52507639)
            4. El 29/08/2017, mulsqi mejoró el récord mundial con un tiempo de [**1:27:49**](https://www.twitch.tv/videos/170696554)
            5. El récord mundial actual lo tiene Tick_Show con un tiempo de [**1:14:29**](www.nicovideo.jp/watch/sm35243230)
            </Steps>
    </TabItem>
