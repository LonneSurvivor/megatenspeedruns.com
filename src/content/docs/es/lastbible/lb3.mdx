## Glitches y Trucos
<details>
<summary>**MP Cheat**</summary>

Como hacerlos:

- Durante cualquier batalla, pulsa el botón de inicio y el menú de batalla cambiará a un menú con tres números. Este es el menú de depuración de in-battle.

- Para cerrarlo, presione nuevamente el botón Inicio.

Características:

Cada uno de los números  puede ser cambiados.

- Presionando el botón A se aumenta el primer número y presionando B se disminuye. Este número representa cada uno de los efectos de ataque en el juego. Al presionar X en un número, puedes ver la animación de ataque del hechizo con esa ID.
- Al pulsar el botón Arriba, el segundo número aumenta y al pulsar el botón Abajo, disminuye. Este número representa el ID de cada encuentro aleatorio, incluyendo cada combate contra jefes. Al pulsar X sobre un número, se puede cambiar el encuentro aleatorio actual al ID seleccionado.
- Al presionar el botón izquierdo, el tercer número aumenta y, al presionar el botón derecho, disminuye. Este número representa el fondo de batalla. Al presionar X, se puede cambiar el fondo al que estés seleccionado.

El demonio de ID 440, cuyo nombre es japonés es [うりね], transliterado a [URINE], Se puede reclutar y puede derrotar de un solo golpe en la mayoría de las peleas debido a que es de nivel 159 y tiene estadísticas defectuosas.
</details>
