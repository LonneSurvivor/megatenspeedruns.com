## Historial de Speedruns
<Steps>
1. Raidou Kuzunoha vs. the Soulless Army fue lanzado en Japón el 30 de Agosto de 2012 y mundialmente el 26 de Septiembre de 2013.
2. La primera run grabada fue realizada el 30/05/2016 por 運の香, se lee como UnnoKou, con un tiempo de [**5:11:12**](https://www.nicovideo.jp/watch/sm28952413) usando la versión de PS2.
3. ExcelitoL mejoró el tiempo usando una consola PS3 console y el PS store de norte-américa a través de la versión 2021.
4. El 11/08/2021 ExcelitoL mejoró el record logrando un tiempo de [**4:31:22**](https://www.youtube.com/watch?v=OKySAA-HPxc).
5. El actual world record le pertenece a Symm con un tiempo de [**4:28:30**](https://youtu.be/m4xMl_L-bn8?si=CFHgheJJulvRsuKe).
</Steps>

## Glitches y Trucos
<details>
<summary>Vsync Delay OFF (PSN only)</summary>
Al usar el sistema de emulación de PS2 en la ps3 y el menú de debug, podes desactivar el Vsync delay. Esto hace que el infame puerto de PSN se ejecute a una velocidad de cuadros mucho más fluida y actualmente está permitido para carreras de velocidad..
</details>
