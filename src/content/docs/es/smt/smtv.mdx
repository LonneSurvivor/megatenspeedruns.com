## Historial de Speedruns
<Tabs>
    <TabItem label="Hard TNE">
        <Steps>
            1. Shin Megami Tensei V fue lanzado mundialmente el 11 de Noviembre de 2021.
            2. La primera run fue hecha el 31/12 de 2021 por romulostx en un tiempo de [**12:40:45**](https://www.twitch.tv/videos/1248653824). Este tiempo luego sería mejorado a [**9:53:27**](https://www.twitch.tv/videos/1449363167)
            3. El 28/05/2022, Shaggy3311 mejoró el tiempo previo de [**9:29:02**](https://www.twitch.tv/videos/1311852027) a [**8:24:14**](https://www.twitch.tv/videos/1582026223)
            4. El 25/11/2022, asapaska mejoró el world record logrando un tiempo de [**8:10:56**](https://www.twitch.tv/videos/1663222487)
            2. El world record actual lo tiene Shaggy3311 con un tiempo de [**7:25:44**](https://www.twitch.tv/videos/1696104648)
        </Steps>
    </TabItem>
    <TabItem label="Safety Any%">
        <Steps>
        1. Shin Megami Tensei V fue lanzado mundialmente el 11 de Noviembre de 2021.
        2. La primera run fue hecha el 27/11/2021 por secretbunnyninja con un tiempo de [**5:09:22**](https://youtu.be/6jNkOEN_jak)
        3. El 12/03/2021, Engiki mejoró el tiempo a [**3:05:45**]. El Vod ya no se encuentra disponible.
        4. 5 días después, sioneus mejoró el record logrando un tiempo de [**2:41:55**](https://www.twitch.tv/videos/1227257096)
        5. 8 días después, Engiki reclamó el record con un tiempo de [**2:40:34**](https://www.twitch.tv/videos/1235172003)
        6. El 22/01/2022, sioneus reclamó el record logrando un tiempo de [**2:32:22**](https://youtu.be/QRTuZuuBN2U)
        7. 1 día después, DarkRedDove impuso un tiempo record de [**2:30:30**](https://www.twitch.tv/videos/1274550863)
        8. En el evento ESA Summer en 2022, sioneus logró un nuevo tiempo con algunas mejoras en la ruta y alcanzó un tiempo de [**2:22:28**](https://www.twitch.tv/videos/1542182099)
        2. El world record actual le pertenece a asapaska con un tiempo de [**2:17:53**](https://youtu.be/uGDf9znO1Gw)
        </Steps>
    </TabItem>
</Tabs>
