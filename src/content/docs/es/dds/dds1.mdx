## Historial de Speedruns
<Tabs>
    <TabItem label="Any%">
        <Steps>
        1. Shin Megami Tensei: Digital Devil Saga fue lanzado originalmente a nivel mundial el 21 de Julio de 2006.
        2. La primera run fue hecha el 22/03/2013 por コッパ con un tiempo de [**6:30:36**](http://www.nicovideo.jp/mylist/36384692).
        3. El 20/01/2016 Kanri mejoró el world record a [**6:17:07**](https://www.twitch.tv/videos/36491163)
        4. El 14/12/2016, ChampionBeef mejoró el record una vez más a [**5:46:41**](https://www.twitch.tv/videos/107477957)
        5. Pronto, PinkPajamas comenzó a realizar mejoras a la ruta, ahora conocidas como la ruta de principiantes o ruta segura, y logró un tiempo de [**4:35:02**](https://www.twitch.tv/videos/247209086) el 08/04/2018 el cual fue el world record por casi 2 años.
        6. El 06/10/2020, Freedom_Pulse superó el world record logrando un tiempo de [**4:32:44**](https://www.twitch.tv/videos/762145103) y el 1 de Junio de 2021, bajó el tiempo a [**4:18:07**](https://www.twitch.tv/videos/1043213075)
        7. El 27/06/2021, mui_14 logró la primera sub 4 horas en un tiempo de [**3:55:49**](https://youtu.be/9uDyoT7ArEQ)
        8. El world record actual le pertenece a mizusima40 con un tiempo de [**3:47:01**](https://www.twitch.tv/videos/1146434213)
        </Steps>
    </TabItem>
    <TabItem label="100%">
        <Steps>
        1. Shin Megami Tensei: Digital Devil Saga fue lanzado originalmente a nivel mundial el 21 de Julio de 2006.
        2. El 01/01/2017, Kanri realizó la primera run con un tiempo de [**31:04:15**](https://www.twitch.tv/videos/111253143)
        3. El world record actual fue hecho el 25/09/2020 por Kanri con un tiempo de [**22:54:45**](https://www.twitch.tv/videos/751575502)
        </Steps>
    </TabItem>
</Tabs>
