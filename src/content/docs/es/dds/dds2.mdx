## Historial de Speedruns
<Tabs>
    <TabItem label="Normal Mode w/o Data Transfer">
        <Steps>
            1. Shin Megami Tensei: Digital Devil Saga 2 fue lanzado a nivel mundial el 17 de marzo de 2007.
            2. La primera run fue hecha por Kanri el 29/06/2016 en un tiempo de [**6:38:39**](https://www.twitch.tv/videos/75217182).
            3. El 08/03/2017, el speedrunner KMB_Chimera mejoró el tiempo establecido siendo este de [**5:25:52**](https://youtu.be/KHk1pWwnWoI)
            4. El 19/04/2017, el speedrunner Otogitune superó el world record a un tiempo de [**5:21:00**](https://youtu.be/eWseqZ659J0)
            5. El 02/09/2017, Kanri recuperó el world record con un tiempo de sub 5 horas, [**4:59:39**](https://www.twitch.tv/videos/171613321)
            4. Después de cási 5 años, el 13/05/2022, Freedom_Pulse superó el world record en un tiempo de [**4:56:38**](https://youtu.be/kVZIZmqB0uM)
            5. El actual world record le pertenece a Jonperi23 con un tiempo de [**4:52:10**](https://youtu.be/Q5RQTGDrL18)
        </Steps>
    </TabItem>
    <TabItem label="Hard Mode, All Bosses">
        <Steps>
        1. Shin Megami Tensei: Digital Devil Saga 2 fue lanzado a nivel mundial el 17 de marzo de 2007.
        2. La primera run fue hecha por Kanri el 09/08/2018 en un tiempo de [**15:59:54**](https://www.twitch.tv/videos/294945452)
        3. El 05/01/2023, Freedom_Pulse superó el world record por dos horas, logrando un tiempo de [**13:52:42**](https://youtu.be/W1v9DgvS6KQ)
        4. En 2023, el reconocido speedrunner que hace la mayoria de rutas, Shaggy3311 comenzó a mejorar la ruta y terminó su primer world record con un tiempo de [**12:20:57**](https://www.twitch.tv/videos/1952253332)
        5. El actual world record le pertenece a Shaggy3311 con un tiempo de [**9:30:49**](https://www.twitch.tv/videos/2345484629)
        </Steps>
    </TabItem>
