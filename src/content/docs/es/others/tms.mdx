## Historial de Speedruns

<Tabs>
<TabItem label="Hard, NG - Wii U">
<Steps>
1. Tokyo Mirage Sessions #FE se lanzó en Japón el 26 de diciembre de 2015.
2. El primer speedrun registrado fue realizado por O-den el 03/05/2017 con un tiempo de [**11:12:55**](https://www.nicovideo.jp/watch/sm31141771)
3. El 29/04/2018 legrandgrand mejoró el récord mundial a un tiempo de [**11:15:36**](https://www.twitch.tv/videos/256836198) con mejoras en la ruta de HeroMystic.
4. El 06/05/2018, HeroMystic recuperó el récord mundial con un tiempo de [**10:30:45**](https://www.twitch.tv/videos/258238938).
5. El 09/05/2018, legrandgrand recuperó el récord mundial con un tiempo de [**9:12:15](https://youtu.be/noBNKezb4TI).
6. El record mundial actual tiene un tiempo de [**8:52:42**](https://youtu.be/HIhQ-qu3Oi8) le pertenece a legrandgrand.
</Steps>
</TabItem>

<TabItem label="Hard, NG - Switch">
<Steps>
1. Tokyo Mirage Sessions #FE Encore fue lanzado en Japón el 17 de enero de 2020.
2. El primer speedrun registrado fue realizado por legrandgrand logrando un tiempo de [**8:29:58**](https://www.twitch.tv/videos/542257073).
3. El record mundial actual le pertenece a legrandgrand con un tiempo de [**7:17:21**](https://www.twitch.tv/videos/549294476).
</Steps>
</TabItem>
</Tabs>
