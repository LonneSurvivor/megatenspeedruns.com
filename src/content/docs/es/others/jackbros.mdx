## Historial de Speedruns

<TabItem label="Beat the Game - Jacks">
<Steps>
1. Jack Bros. fue lanzado en Japón el 29 de septiembre de 2005.
2. El 08/01/2017 la primera run fue realizada por SomeGirl en un tiempo de [**25:35**](https://youtu.be/K15Dcpv4vRw)
3. El 21/04/2017, Witchs_Hex logró un nuevo world record en un tiempo de [**23:41**](https://youtu.be/ZbuK4J1MMoU)
4. 3 años después, el 21/11/2020, swordsmankirby venció el antiguo world record, logrando un tiempo de [**23:23**](https://youtu.be/PhbqjHWz2AY)
5. El actual world record le pertenece a SmoothColaPepsi con un tiempo de [**22:34**](https://youtu.be/i8wqf9478tk)

</details>

</details>
