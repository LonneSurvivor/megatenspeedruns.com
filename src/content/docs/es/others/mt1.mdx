## Historial de Speedruns
<Steps>
1. Digital Devil Story: Megami Tensei fue lanzado en Japón el 11 de Septiembre de 1987.
2. La primera grabación de un speedrun encontrada fue subida por [めし太郎](https://www.nicovideo.jp/user/2729422?rf=nvpc&rp=watch&ra=video_information) el 06/07/2011 en un tiempo de [**9:16:03**](https://www.nicovideo.jp/watch/sm14942670)
3. El 05/04/2013, biim subió una run con un tiempo de [**5:04:05**](https://www.nicovideo.jp/watch/sm20530524).
4. El 29/07/2013, runner sinri subió un nuevo world record con un tiempo de [**4:25:02**](https://www.nicovideo.jp/watch/sm21472094).
5. El 15/02/2014, runner dasi subió una mejora del world record en un tiempo de [**3:18:06**](https://www.nicovideo.jp/watch/sm22899800).
6. El récord mundial actual parece ser un tiempo de [**3:13:04**](https://www.twitch.tv/videos/2357513079?collection=RjiGK9oJcBZ6PQ) el 19/01/2025 by Goro_Itsumi . Este speedrun utiliza el *Medical Loop*.
</Steps>
