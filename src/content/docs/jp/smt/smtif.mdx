## 歴史 

<Tabs>
    <TabItem label="SFC版 - ユミ">
        <Steps>
            1. 真・女神転生if。。。は１９９４年１０月２８日に発売されました。
            2. 初めて録画した自己ベストはコロスケさんに達成されました。時間間は[**3時間02分12**](https://www.nicovideo.jp/watch/sm4278579)でした。
            3. ２０１８年１１月１８日、Symmsさんは新世界記録を達成しました。時間間は [**1時間52分34**](https://youtu.be/jpRJI88N1Ak)でした。
            4. ２０１９年０６月０１日、水島さんは１時間５０ギリを超えて記録を更新しました。時間間は[**1:30:16**](https://www.twitch.tv/videos/382656890)でした。
            5. ２０２１年０１月１４日、Symmさんはもう一度世界記録を取りました。時間間は[**1時間30分15**](https://youtu.be/XJUBxrIB-0o)でした。この記録は実行再現を使用しました。
            6. 世界記録は水島さんの[**１時間２７分４５**](https://youtu.be/3BzRy-mCDPo).
        </Steps>
    </TabItem>
    <TabItem label="SFC版 - アキラ">
        <Steps>
            1. 真・女神転生if。。。は１９９４年１０月２８日に発売されました。
            2. 初めて録画した自己ベストはドミネーターに達成されました。時間間は[**2時間09分51**](https://youtu.be/XJUBxrIB-0o)でした。
            3. ２０１７年０４月２３日にkakkunさんは記録を更新しました。時間間は[**2時間00分05**](https://youtu.be/w-e6EiCBUi8)でした。
            4. ２０１９年０７月０１日に水島さんは新世界記録を達成しました。 時間間は[**1時間50分8**](https://www.twitch.tv/videos/405448859)でした。
            5. ２０２３年０３月１７日、ぴぴおさんはチャートを更新して４年間に生き残した記録を更新しました。時間間は[**1時間45分50**](https://www.twitch.tv/videos/1767631900)でした。
            6. ２０２４年１１月０９日、水島さんはいきなり記録を取り直しました。現在の世界記録は[**1時間45分05**](https://www.twitch.tv/videos/2297346734)です。
        </Steps>
    </TabItem>
    <TabItem label="PS版 - ユミ">
        <Steps>
            1. 真・女神転生if。。。は２００２年１２月２６日に発売されました。
            2  PS1版『If...』の最初の記録は、2011年3月4日、TA539がチャーリー編を難易度EASY、制限なしで完走したもので、時間間は[**1時間39分14**](https://www.nicovideo.jp/watch/sm13781864)でした。
            3. 最初のユミ編の記録は、えくいさんが難易度EASY、制限なしで完走したもので、時間間は[**1時間12分34**](http://www.nicovideo.jp/watch/sm33552720)でした。
            3. ２０２１年０８月０９日, R-0109さんはユミ編のNORMAL難易度、裏技・バグなしカテゴリーを完走しました。時間間は[**1時間30分13**](https://www.twitch.tv/videos/1137351030)でした。
            4. ２０２３年０２月１５日、ToxicTTさんはユミ編のEASY難易度、裏技・バグなしカテゴリーを完走して新記録を達成しました。時間間は[**1時間21分15**](https://www.twitch.tv/videos/1737221310)でした。
        </Steps>
    </TabItem>
    <TabItem label="PS版 - アキラ">
        <Steps>
            1. 真・女神転生if。。。は２００２年１２月２６日に発売されました。
            2. ２０１８年０７月１９日にえくいさんは初めてEASY難易度のRTAを完走しました。時間間は[**1時間16分09**](http://www.nicovideo.jp/watch/sm37411190)でした。
        </Steps>
    </TabItem>
</Tabs>
