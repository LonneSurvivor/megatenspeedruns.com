## Speedrun History
<Tabs>
    <TabItem label="PS1 - SEBEC">
        <Steps>
            1. Relevations: Persona was released in Japan on September 20, 1996.
            2. The current world record holder is Supergamer57 with a time of [**8:12:35**](https://www.twitch.tv/videos/42673047)
        </Steps>
    </TabItem>
    <TabItem label="PS1 - Snow Queen Quest">
        <Steps>
            1. Relevations: Persona was released in Japan on September 20, 1996.
            2. The current world record holder is Tick_Show with a time of [**4:15:24**](https://nico.ms/sm35227898?ref=thumb_watch)
        </Steps>
    </TabItem>
    <TabItem label="PSP - SEBEC">
        <Steps>
            1. Shin Megami Tensei: Persona was released in Japan on April 29, 2009.
            2. The first recorded run was done by Liv based on Supergamer57's PS1 SEBEC route which achieved a time of [**6:05:32**](https://youtu.be/7OyKC3M5TRs)
            3. On 2019/06/12, Tick_Show developed the Mabufu Stone route and achieved a time of [**3:59:53**](www.nicovideo.jp/watch/sm35250359)
            4. On 2019/07/26, runner escargoon achieved the first sub 3h run with a time of [**2:54:13**](https://youtu.be/VXMjtP7fWMo)
            5. The current console world record holder is Tick_Show with a time of [**2:51:05**](https://youtu.be/HWlsj6S-7Xw) and the lowest ever achieved time, which was done on emulator, is held by Fenr1r74 with a time of [**2:44:48**](https://www.twitch.tv/videos/1825596953)
        </Steps>
    </TabItem>
    <TabItem label="PSP - Snow Queen Quest">
        <Steps>
            1. Shin Megami Tensei: Persona was released in Japan on April 29, 2009.
            2. The first recorded run was done by escargoon on 2013/12/24, completing the True Ending, with a time of [**1:56:33**](https://youtu.be/ijDyyVHEmro). True Ending is the less popular category and its fastest time is now held by Tick_Show with a time of [**1:48:27**](www.nicovideo.jp/watch/sm35246032)
            3. The first recorded Bad Ending run was done by Symm on 2016/03/05 with a time of [**1:30:50**](https://www.twitch.tv/videos/52507639)
            4. On 2017/08/29, mulsqi improved the world recorded with a time of [**1:27:49**](https://www.twitch.tv/videos/170696554)
            5. The current world record is held by Tick_Show with a time of [**1:14:29**](www.nicovideo.jp/watch/sm35243230)
            </Steps>
    </TabItem>
    
</Tabs>
